1  Centiloid

Methods for Aβ Centiloid Standardization

Authors Amal Tiss, Ph.D.1, Julie Price, Ph.D.1, and Charles Laymon, Ph.D.2

Affiliations

  1. Massachusetts General Hospital, Co-Investigators on ABC-DS
  2. University of Pittsburgh, Co-Investigators on ABC-DS

Summary (or Abstract): Centiloid Standardization

PET measures of amyloid-beta (Aβ) load determined using two different Aβ imaging agents, [11C]PiB (PiB) and [18F]Florbetapir (FBP) were standardized by Centiloid (CL) scaling.[1] Regional PiB SUVR and FBP SUVR outcomes were linearly transformed to a CL scale based on PiB as reference standard. Centiloid values of 0 and 100 respectively correspond to “high-certainty” Aβ-negative young healthy control (YC) subjects (≤45 years) and “typical” Aβ-positive early/mid-stage Alzheimer’s disease (AD) patients.

1.1 Method

Original Methods: The PET Image data were acquired over 50-70 minute post-injection interval[2] and reconstructed into four, 5 minute frames. PET images were first preprocessed to correct for motion between frames. Images were realigned to the first image frame and averaged (over 50min-70min), to generate a mean realigned FBP image. Each participant’s MRI and mean PET images were then manually reoriented to the anterior commissure/ posterior commissioner line using Statistical Parametric Mapping 8 (SPM8). These images were then coregistered and normalized into MNI152 space according to the Klunk et al. Standard Centiloid methodology.[1] Standard Centiloid Volume of Interests (VOIs) were downloaded from the Global Alzheimer’s Association Interactive Network (GAAIN) website (www.GAAIN.org). A𝛽-PET SUVR values (50-70 min post injection mark) were determined using the standard Centiloid cortex VOI that that was generated from areas with high A𝛽-load in Alzheimer’s: frontal, temporal, parietal cortices, precuneus, anterior striatum, and insular cortex. Whole cerebellum served as the reference regions in the SUVR calculations for both PiB and FBP. No Partial volume correction was performed. The PiB and FBP data were converted to Centiloids, using previously published methods.[1,3]

Update on February 10, 2026 The PiB and FBP data were originally processed using the standard CL process described above[1,3] with whole cerebellum as reference and use of standard set of CL regionsof- interest,[1] with SUVR values determined from data acquired 50-70 minutes after tracer injection.

As reported previously,[4] reference region selection has a strong influence on the detection of FBP longitudinal change and reference regions that include subcortical white matter (WM) resulted in change measures that are more accurate than those determined with cerebellar or pons data alone. Therefore, FBP SUVR values were also computed using 2 types of WM reference regions: [1] the eroded cerebral WM and [2] a composite region composed of the eroded cerebral WM, the whole cerebellum, and the brainstem.

An eroded cerebral WM mask was generated using the white matter probability map in the native MR space that was generated during the MNI normalization step of the standard centiloid pipeline. This map underwent Gaussian smoothing (8 mm) and thresholding (90%) to produce an eroded WM mask in the native MR space.[5] This was normalized to MNI template space using the same transformation applied to the MR image as part of the standard centiloid pipeline. Only cerebral WM voxels were included (i.e. excluded WM in cerebellum and brainstem). CL values were computed using higher level stepwise CL calibration[1] based on the Navitsky GAAIN dataset,[3] the standard CL cortex target regions, and the WM reference regions. The PiB images continue to be processed using the whole cerebellum as the reference region.
